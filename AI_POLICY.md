# 🤖 AI Policy

## Our Position

Planify is **open to the use of AI-assisted tools** in development. We believe AI can be a valuable tool for improving code quality, productivity, and accessibility for contributors.

## Guidelines

- **AI is a tool, not a replacement.** Contributors are fully responsible for any code they submit, whether written manually or with AI assistance.
- **Human review is required.** All contributions go through the same review process regardless of how they were written. A human maintainer will always review and approve changes before merging.
- **Understand what you submit.** If you use AI to generate or modify code, make sure you understand it, can explain it, and can maintain it.
- **No AI-generated content without review.** Do not submit AI output blindly. Test it, verify it, and ensure it follows the project's coding standards and architecture.
- **Transparency is appreciated.** While not mandatory, mentioning that AI tools were used in your contribution is welcome and helps maintain trust within the community.

## For Users

Some contributors and maintainers may use AI-assisted tools during development. All code merged into the project has been reviewed and approved by human maintainers.

## Summary

We don't restrict the use of AI tools — we trust our contributors to use them responsibly. The quality bar for contributions remains the same regardless of how the code was written.
